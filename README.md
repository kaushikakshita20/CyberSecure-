# 🛡️ CyberSecure – Cybersecurity Awareness Website

CyberSecure is a responsive cybersecurity awareness and learning platform developed using HTML, CSS, and JavaScript.
The website helps users understand cyber threats, improve online safety practices, and develop secure digital habits through interactive tools and educational content.

---

# 🎯 Project Objective

- 🔐 Spread cybersecurity awareness
- 🧠 Help users learn safe online practices
- 🛡️ Provide interactive cybersecurity tools
- ⚡ Improve digital safety knowledge

---

# 🚀 Features

## 🏠 Home Page

- Attractive hero section
- 📊 Animated cyber attack counter
- Explanation of CIA Triad:
  - 🔒 Confidentiality
  - ✔️ Integrity
  - 🌐 Availability

---

## ⚠️ Cyber Threats Section

Includes information about:

- 🎣 Phishing
- 💀 Ransomware
- 💉 SQL Injection
- 🌊 DDoS Attacks
- ⌨️ Keyloggers
- 🔓 Brute Force Attacks

Each threat contains:
- 📖 Description
- 🛡️ Prevention Tips
- 🚨 Severity Tags
